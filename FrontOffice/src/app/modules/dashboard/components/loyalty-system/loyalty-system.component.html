Loyalty System

Here you can see how the loyalty system works and how the discounts are applied!

How can I aquire points?

For each transaction that a user makes (buying or selling books), the user receives 1 point + (the total price of the transaction * 15%). For example, if a user makes a purchase worth 50€, that user will get 1 + (50 * 15%) = 8 points.

How can I use my points?

The points are used to get discounts in purchases, on the checkout screen there is a checkbox to apply the discount if certain criteria are met. Discounts cost 10 points and the user receives a % of discount based on their age group.

To be able to get the discount the user needs to meet a criteria from the second group of the table (Number of purchased and/or number of sold books). For example:

  • If the user has 10 points, is an adult, makes a transaction of 50 € and has bought 10 books in the past but didin't sell any books, he will receive a discount of 25%.
  • If the user is an adult has less than 10 points, makes a transaction of 50 € and has bought 10 books in the past but didin't sell any books, he won't be able to receive discounts.
